# RACE — 通用均衡

**适用场景：** 最通用的框架，适合大多数日常任务。在简洁和完整之间取得了最佳平衡。

**参数数量：** 4

## 模板

```
## RACE 框架

**Role（角色）**：[你希望AI扮演什么角色，如：资深分析师/前端工程师/产品经理]
**Action（行动）**：[需要完成的具体任务，描述越具体越好]
**Context（上下文）**：[背景信息、相关情况、限制条件、目标受众]
**Expectation（期望）**：[输出格式、风格、长度、质量标准]
```

## 示例

```
## RACE 框架

**Role（角色）**：资深股票分析师，专注于A股科技板块
**Action（行动）**：分析海康威视（002415）当前的估值水平，给出买入/持有/卖出的建议
**Context（上下文）**：
- 海康威视是安防龙头，但面临AI转型和海外市场的不确定性
- 当前PE约20倍，处于历史低位区间
- 用户偏好价值投资，持有周期1-3年
**Expectation（期望）**：
- 300-500字精炼分析
- 必须包含具体估值数据
- 结论明确，不模棱两可
```
