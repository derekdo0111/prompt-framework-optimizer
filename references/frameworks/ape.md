# APE — 行动导向

**适用场景：** 轻量任务，比TAC多一层目的说明。适合有明确目的但不复杂的任务。

**参数数量：** 3

## 模板

```
## APE 框架

**Action（行动）**：[要执行的具体任务]
**Purpose（目的）**：[为什么要做这件事，背景和动机]
**Expectation（期望）**：[期待什么样的输出结果，包含格式、风格、长度等]
```

## 示例

```
## APE 框架

**Action（行动）**：用Python写一个函数，读取CSV文件并计算每列的平均值
**Purpose（目的）**：用于批量处理财务数据文件，需要快速得到统计摘要
**Expectation（期望）**：
- 返回一个字典，key为列名，value为平均值
- 自动跳过非数值列
- 处理缺失值（NaN）
```
