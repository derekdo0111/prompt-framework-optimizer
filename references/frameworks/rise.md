# RISE — 步骤清晰

**适用场景：** 需要分步骤完成的复杂任务，每一步有明确的输入输出。适合数据分析、报告撰写、代码开发等流程化任务。

**参数数量：** 4

## 模板

```
## RISE 框架

**Role（角色）**：[AI扮演的专业角色]
**Input（输入）**：[提供的原始材料/数据/信息，可附上具体内容]
**Steps（步骤）**：[希望AI按什么顺序处理]
    1. [第一步：理解/分析/拆解输入]
    2. [第二步：核心处理/计算/推理]
    3. [第三步：输出/总结/呈现结果]
**Expectation（期望）**：[每步期望的产出标准，以及最终交付物格式]
```

## 示例

```
## RISE 框架

**Role（角色）**：资深财务分析师，精通A股财报分析

**Input（输入）**：
以下为某公司近三年财务数据摘要：
- 2023年营收：500亿，净利润：45亿，经营现金流：60亿
- 2024年营收：520亿，净利润：38亿，经营现金流：42亿
- 2025年营收：550亿，净利润：52亿，经营现金流：48亿

**Steps（步骤）**：
    1. 计算三年的营收增速、净利润率、现金流/利润比率
    2. 分析利润质量（经营现金流是否持续覆盖净利润）
    3. 识别潜在风险点（如2024年利润下滑原因）
    4. 给出综合财务健康度评分（1-10分）

**Expectation（期望）**：
- 以结构化表格呈现计算结果
- 每个风险点附带具体数据支撑
- 最终结论不超过200字
```
