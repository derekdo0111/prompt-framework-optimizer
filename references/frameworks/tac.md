# TAC — 极简三要素

**适用场景：** 极其简单的任务，一句话就能说清，不需要背景铺垫。如快速问答、简单查询、日常闲聊式任务。

**参数数量：** 3

## 模板

```
## TAC 框架

**Task（任务）**：[一句话说清楚要做什么]
**Action（动作）**：[具体行动方式，如：直接回答/列举/总结]
**Goal（目标）**：[期望达成的结果，一句话]
```

## 示例

```
## TAC 框架

**Task（任务）**：解释什么是DCF估值法
**Action（动作）**：用通俗易懂的语言，举一个生活化的例子
**Goal（目标）**：让完全不懂金融的人也能理解核心概念
```
