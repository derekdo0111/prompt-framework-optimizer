# ERA — 角色优先

**适用场景：** 角色扮演类任务，强调AI应以特定身份和视角来回答问题。适合创意写作、模拟对话、风格模仿等。

**参数数量：** 3

## 模板

```
## ERA 框架

**Expectation（期望）**：[最终想要什么效果或产出，先明确终点]
**Role（角色）**：[AI应扮演的身份、专业视角、性格特征]
**Action（行动）**：[基于上述角色和期望，具体要做什么]
```

## 示例

```
## ERA 框架

**Expectation（期望）**：产出一篇小红书风格的产品推荐文案，能激发购买欲望，带emoji和话题标签
**Role（角色）**：你是小红书美妆博主，25岁，风格亲切接地气，粉丝喜欢你的真实测评
**Action（行动）**：写一篇防晒霜测评文案，包含使用感受、优缺点对比、购买建议
```
